This the Read Magnon Config component and is used to read the configurations from Magnon Eg Plus configuration.
To fetch the values and fill against the values below, configure the Magnon Eg Plus configuration by clicking here.

Address: ${magnonConfig.getMagnonIndiaAddress}
Contact Number: ${magnonConfig.getMagnonIndiaContactNumber}
Employee Name: ${magnonConfig.getMagnonEmployeeName}
Employee Location: ${magnonConfig.getMagnonEmployeeLocation}

Highlights of this implementation:
  • The component makes use of the latest OSGI annotations over the obsolete felix annotations to create the component(s)/service(s) and the configurations.
  • The Sling Model Exporter framework has been used instead of the plain old Sling Models.
    This provides the ability to get the business view of the component and not only the authored properties.
    In this case, the authored text can be large article but its business view returns truncated summary of the same.
